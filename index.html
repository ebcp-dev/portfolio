FORUM'd

FORUM'd is an online forum where users can register for accounts, submit either link posts or text posts, and comment on posts. Node.js/Express is used in the backend REST API. User and post data are stored in a MongoDB database. React is used to build the view layer of the app with Bootstrap for styling the UI. Passport.js and Bcrypt handle user authentication and password hashing respectively.
